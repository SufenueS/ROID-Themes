# ROID Themes

Official theme registry for [ROID](https://github.com/SufenueS/ROID) by Sufenue.

ROID fetches this repo's `index.json` and downloads each listed theme into the user's local `themes/` folder. Themes appear in the Customize page → Themes gallery.
